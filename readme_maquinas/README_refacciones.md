# Refacciones – Categorías (sin catálogo)

Esta entrega actualiza `refacciones.html` para mostrar **categorías** de refacciones (sin listado de SKUs),
con 14 familias principales y filtros por categoría.

## Cambios
- Se reemplazó el grid de items por **14 categorías** (cards).
- Se actualizó la barra de filtros para esas 14 categorías.
- Se eliminaron los `>` huérfanos (entidades `&gt;`) que aparecían en el HTML.

## Nota
Si querés, el siguiente paso es que al hacer click en una categoría:
- abra WhatsApp con un mensaje prellenado (ej: “Quiero cotizar Hooks para Brother S-7180”)
- o que abra una subpágina por categoría (sin cambiar tu estilo).

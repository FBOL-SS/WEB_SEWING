# Sewing Solution – Maquinaria (Catálogo)

Esta entrega actualiza **maquinaria.html** para que el catálogo deje de mostrar modelos random y use únicamente los modelos definidos por categoría.

## Qué se cambió
- Catálogo reconstruido por categorías: Rectas, Doble Aguja, Overlock, Coverstitch, Bartack, Ojal, Automáticas y CNC.
- Barra de filtros actualizada (categorías + marcas).
- Se eliminaron caracteres huérfanos `>` que estaban apareciendo.
- Imágenes cambiadas a rutas locales (ya no se usa `static.photos`).

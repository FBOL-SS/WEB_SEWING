# Brand Logos Folder

Coloca aquí los logos de las marcas que distribuyes.

## Archivos necesarios:

| Archivo | Descripción |
|---------|-------------|
| `brother.png` | Logo de Brother Industrial |
| `pegasus.png` | Logo de Pegasus |
| `yamato.png` | Logo de Yamato |
| `supreme.png` | Logo de Supreme |

## Especificaciones recomendadas:

- **Formato:** PNG con fondo transparente
- **Tamaño:** 200px de ancho x 80px de alto (aproximado)
- **Resolución:** 72-150 dpi
- **Color:** Preferiblemente versión horizontal

## Notas:

- Si no hay imagen disponible, el sitio mostrará automáticamente el nombre de la marca en texto
- Los logos se muestran en escala de grises suave y se iluminan al pasar el mouse
- Mantén los nombres de archivo en minúsculas como se indica arriba
